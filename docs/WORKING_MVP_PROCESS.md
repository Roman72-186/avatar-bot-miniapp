# Рабочий MVP-процесс без Т-Банк

Дата: 2026-05-06

## Что отключено

Т-Банк временно не участвует в пользовательском процессе:

- нет кнопки `Карта / СБП` в Mini App;
- frontend не вызывает `create-tbank-payment`, `check-tbank-payment`;
- deploy config и `docker-compose.yml` не требуют `TBANK_*`;
- `manifest.json` импортирует только текущие рабочие workflow.

## Что работает сейчас

Frontend недвижимости использует уже активные n8n endpoints:

> Текущий серверный HTTPS-маршрут n8n работает через `https://n8n.creativeanalytic.ru:8443/webhook`, потому что порт `443` занят Marzban.

| UI-режим | Endpoint | Стоимость в UI | Что происходит |
|---|---|---:|---|
| AI-ремонт | `generate` | 8 кредитов | фото отправляется с real-estate prompt для визуализации ремонта |
| Улучшить фото | `generate-enhance` | 8 кредитов | улучшение фото через рабочий enhance workflow |
| Видео объекта | `generate-video` | 155 кредитов | видео из первого фото, последнее фото передается как last frame |
| Текст объявления | локально | 0 кредитов | черновик текста строится из object info без backend |
| Полный пакет | `generate-video` + локальный текст | 155 кредитов | видео через рабочий workflow, текст собирается локально |

## Ограничения MVP

- `AI-ремонт` пока списывается как старый `generate` workflow, то есть 8 кредитов.
- `Видео объекта` использует существующий `generate-video`, который технически принимает одно основное фото и optional last frame.
- `Текст объявления` сейчас шаблонный, без LLM backend.
- Отдельные `generate-real-estate-*` workflow оставлены как future templates и не подключены к рабочему процессу.

## Проверка

1. Открыть Mini App.
2. Проверить, что пополнение показывает только Telegram Stars.
3. Пополнить баланс Stars.
4. Запустить `AI-ремонт` на одном фото.
5. Запустить `Улучшить фото`.
6. Запустить `Видео объекта` на 3 фото.
7. Сгенерировать `Текст объявления`.
8. Открыть историю и проверить image/video/text previews.
